# How to run NUERALLY Website

Since you do not have Node.js or Python installed/configured in your PATH, you can view the website by opening the HTML file directly.

## Option 1: Double Click
1. navigate to `c:\Users\token\Nuerally Token\` in File Explorer.
2. Double click `index.html`.

## Option 2: Install a Local Server (Recommended)
For the full experience (and to avoid CORS issues with some external resources), install Node.js:
1. Download from [https://nodejs.org/](https://nodejs.org/)
2. Run `npx serve .` in this directory.
